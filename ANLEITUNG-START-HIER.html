🦷 Zahngut App - Fertig in 5 Minuten!

Einfache Schritt-für-Schritt-Anleitung

1

📥 Dateien herunterladen

Laden Sie diese 4 Dateien herunter:

📄
index.html
(Hauptdatei)
📱
manifest.json
(PWA Config)
⚙️
sw.js
(Service Worker)
🎨
icon-generator.html
(Icon Tool)
2

📂 Dateien ersetzen/hochladen

⚠️ Wichtig: Benennen Sie index-firebase-ready.html um zu index.html bevor Sie hochladen!

So sieht Ihre Ordnerstruktur aus:

  • 📁 zahngut-app/
  •     📄 index.html ← NEU (war index-firebase-ready.html)
  •     📄 manifest.json ← NEU
  •     📄 sw.js ← NEU
  •     📄 admin-firebase.html ← haben Sie schon
  •     📄 admin-login.html ← haben Sie schon
  •     📄 firebase-config.js ← haben Sie schon
  •     📄 firebase-service.js ← haben Sie schon
  •     📄 [andere Dateien...]
  •     📁 icons/ ← Ordner erstellen!
3

🎨 Icons generieren

App-Icons erstellen:

  1. Öffnen Sie icon-generator.html im Browser
  2. Klicken Sie auf "Icons generieren"
  3. Klicken Sie auf "Alle Icons herunterladen"
  4. Speichern Sie alle Icons im Ordner icons/
💡 Tipp: Die Icons werden automatisch mit dem Zahngut-Logo generiert!
4

🚀 Testen

Prüfen Sie, ob alles funktioniert:

📱 Lokal testen:

# Python (wenn installiert): python3 -m http.server 8080 # Oder mit Node.js: npx http-server # Dann öffnen: http://localhost:8080

🌐 Online testen:

  1. Laden Sie alle Dateien auf Ihren Server hoch
  2. Öffnen Sie Ihre Domain im Browser
  3. Testen Sie auf dem Handy!
5

📲 Als App installieren

Die App auf Ihrem Handy installieren:

iPhone (iOS):

  1. Öffnen Sie Safari
  2. Gehen Sie zu Ihrer App-URL
  3. Tippen Sie auf das Teilen-Symbol ⬆️
  4. Wählen Sie "Zum Home-Bildschirm"
  5. Name eingeben und "Hinzufügen" tippen

Android:

  1. Öffnen Sie Chrome
  2. Gehen Sie zu Ihrer App-URL
  3. Chrome zeigt automatisch "App installieren"
  4. Oder: Menü (3 Punkte) → "App installieren"
🎉 Fertig! Ihre App ist jetzt installierbar und funktioniert offline!
